# 采购与委外采购业务流程说明

## 一、销售订单与采购来源

系统中的采购业务来源于销售订单。

销售订单可以拆分为多个购销订单，形成分销购销订单。销售订单会进一步触发以下几类采购流程：

1. 现购采购订单
2. 标准采购订单
3. 委外采购订单

---

# 二、现购采购流程

## 1. 现购采购订单

现购采购订单创建后，进入付款流程。

流程如下：

```text
现购采购订单
→ 付款申请单
→ 付款完成
→ 收料通知单
```

随后进入收货与检验流程。

如果发生退货：

```text
现购采购订单
→ 采购退料单
```

---

# 三、标准采购流程

## 1. 标准采购订单

标准采购订单创建后：

```text
标准采购订单
→ 收料通知单
→ 检验单
```

---

## 2. 检验结果处理

### （1）检验合格

如果检验结果为合格：

```text
检验单
→ 合格
→ 收料通知单
→ 采购入库单
```

---

### （2）检验不合格

如果检验结果为不合格：

```text
检验单
→ 不合格
→ 不良品处理单
```

不良品处理后：

```text
不良品处理单
→ 采购入库单
```

或者：

```text
不良品处理单
→ 采购退料单
```

---

# 四、采购入库与付款流程

采购入库后：

```text
采购入库单
→ 应付单
→ 付款申请单
```

形成完整付款流程。

---

# 五、委外采购流程

## 1. 委外采购订单

委外采购订单生成后：

```text
委外采购订单
→ 采购订单
```

采购订单审核后：

```text
采购订单
→ 委外领料单
```

同时进入收料流程：

```text
采购订单审核后
→ 收料通知单
→ 采购入库单
```

---

# 六、费用采购流程

费用类采购流程如下：

```text
费用订单
→ 收料通知单
→ 应付单
```

属于无需库存入库的费用型采购。

---

# 七、采购退料流程

## 1. 入库后发现不良

如果采购入库后检验发现不良品：

```text
采购入库单
→ 检验有不良品
→ 退料申请单
→ 采购退料单
```

---

## 2. 退料后的财务处理

采购退料后：

```text
采购退料单
→ 扣款
→ 应付单
```

---

# 八、补货流程

采购退料后如果需要补货：

```text
采购退料单
→ 补货
→ 原对应采购订单
```

然后重新执行：

```text
原对应采购订单
→ 收料通知单
→ 检验单
→ 采购入库单
→ 应付单
→ 付款申请单
```

形成完整补货闭环。

---

# 九、完整业务主链路

## 标准采购主流程

```text
销售订单
→ 标准采购订单
→ 收料通知单
→ 检验单
→ 采购入库单
→ 应付单
→ 付款申请单
```

---

## 委外采购主流程

```text
销售订单
→ 委外采购订单
→ 采购订单
→ 委外领料单
→ 收料通知单
→ 采购入库单
→ 应付单
```

---

## 不良品处理流程

```text
检验单
→ 不良品处理单
→ 采购退料单
→ 扣款
→ 应付单调整
```

---

## 补货闭环流程

```text
采购退料单
→ 原对应采购订单
→ 收料通知单
→ 检验单
→ 采购入库单
→ 应付单
→ 付款申请单
```

---

# 十、业务对象说明

| 单据名称   | 说明          |
|--------|-------------|
| 销售订单   | 客户销售需求来源    |
| 分销购销订单 | 销售订单拆分后的购销单 |
| 标准采购订单 | 常规采购业务      |
| 委外采购订单 | 委托外部加工采购    |
| 采购订单   | 实际执行采购      |
| 收料通知单  | 通知仓库或质检收货   |
| 检验单    | 质检结果记录      |
| 不良品处理单 | 不合格品处理      |
| 采购入库单  | 采购物料正式入库    |
| 应付单    | 财务应付款记录     |
| 付款申请单  | 付款审批流程      |
| 退料申请单  | 申请退回不良物料    |
| 采购退料单  | 实际退货单据      |
| 委外领料单  | 委外加工发料      |
| 费用订单   | 非库存费用采购     |
